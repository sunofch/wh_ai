# 机械备件知识库
## Mechanical Parts Reference

本文档包含仓库机械区（Mech1、Mech2）的备件信息，用于AGV调度指令解析。

---

## 轴承（Bearing）
- **中文名称**: 轴承 / 滚动轴承 / 轴承件
- **常用别名**: 轴承座、滚珠轴承、推力轴承
- **所属储区**: Mech1、Mech2（机械区）
- **储位格式**: MechX_RY_BZ（如 Mech1_R2_B5）
- **常见操作指令**:
  - 出库: "需要N个轴承"、"取N个轴承"、"领取轴承"
  - 入库: "补充轴承"、"入库N个轴承"、"补货轴承"
  - 调库: "调拨轴承"、"转移轴承到"

## 齿轮（Gear）
- **中文名称**: 齿轮 / 传动齿轮 / 减速齿轮
- **常用别名**: 齿轮件、驱动齿轮
- **所属储区**: Mech1、Mech2
- **常见操作指令**:
  - 出库: "需要N个齿轮"、"取齿轮"
  - 入库: "补货齿轮"、"入库齿轮"

## 液压泵（Hydraulic Pump）
- **中文名称**: 液压泵 / 油泵
- **常用别名**: 液压油泵、柱塞泵、齿轮泵
- **所属储区**: Mech1、Mech2
- **常见操作指令**:
  - 出库: "需要液压泵"、"取液压泵"、"领液压泵"
  - 入库: "入库液压泵"

## 联轴器（Coupling）
- **中文名称**: 联轴器 / 弹性联轴器
- **常用别名**: 联轴节、联轴件
- **所属储区**: Mech1、Mech2
- **常见操作指令**:
  - 出库: "取联轴器"、"需要联轴器"

## 制动器（Brake）
- **中文名称**: 制动器 / 刹车
- **常用别名**: 制动件、制动装置、刹车器
- **所属储区**: Mech1、Mech2
- **常见操作指令**:
  - 出库: "需要制动器"、"取刹车"
  - 入库: "入库制动器"

---

## 操作分类说明
- **出库**: 关键词包含"需要、取、领取、拿、出库"，action_required = "出库"
- **入库**: 关键词包含"补货、入库、补充、存入"，action_required = "入库"
- **调库**: 关键词包含"调拨、转移、移库"，action_required = "调库"
- **紧急**: 关键词包含"紧急、急、马上、立刻"，is_urgent = true
