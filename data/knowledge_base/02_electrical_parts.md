# 电气备件知识库
## Electrical Parts Reference

本文档包含仓库电气区（Elec1、Elec2）的备件信息，用于AGV调度指令解析。

---

## 三相异步电动机（Three Phase Induction Motor）

- **中文名称**: 三相异步电动机 / 三相电机 / 异步电机
- **英文名称**: Three Phase Induction Motor
- **英文缩写**: TPIM
- **型号示例**: Y160M-4-11kW-ABB、Y132M-4-7.5kW-WEG、MS160M-4-11kW-SIEMENS
- **安装设备**: 输送线驱动电机、提升机主驱动、液压泵站电机
- **规格参数**:
  - 额定功率: 11kW / 额定电压: 380V / 额定电流: 22.6A
  - 同步转速: 1500rpm / 效率等级: IE3 / 防护等级: IP55

---

## 漫反射光电传感器（Diffuse Photoelectric Sensor）

- **中文名称**: 漫反射光电传感器 / 光电开关 / 光电传感器
- **英文名称**: Diffuse Reflection Photoelectric Sensor
- **英文缩写**: DRPS
- **型号示例**: E3Z-D61-2M-OMRON、WL4-3P2430-SICK、OBT200-18GM60-E5-V1-PEPPERL
- **安装设备**: 输送线物品检测、AGV定位辅助、货架存取确认
- **规格参数**:
  - 检测距离: 0.1-2m / 光源: 红外LED 880nm
  - 输出类型: NPN集电极开路 / 响应时间: 1ms / 防护等级: IP67

---

## 变频调速器（Variable Frequency Drive）

- **中文名称**: 变频调速器 / 变频器 / 变频驱动器
- **英文名称**: Variable Frequency Drive
- **英文缩写**: VFD
- **型号示例**: ACS580-039A-ABB、VLT-AQUA-FC202-DANFOSS、FR-E840-MITSUBISHI
- **安装设备**: 输送线调速控制柜、提升机变频控制系统、水泵变频节能改造
- **规格参数**:
  - 额定功率: 18.5kW / 输入电压: 380-480V 3相
  - 输出频率: 0-300Hz / 控制方式: 矢量控制 / 防护等级: IP21

---

## 可编程逻辑控制器（Programmable Logic Controller）

- **中文名称**: 可编程逻辑控制器 / PLC / 可编程控制器
- **英文名称**: Programmable Logic Controller
- **英文缩写**: PLC
- **型号示例**: CPU1214C-DC-SIEM、Q04UDVCPU-MITSUBISHI、CP1H-X40DT-D-OMRON
- **安装设备**: 自动化控制主柜、AGV调度控制站、输送线顺序控制系统
- **规格参数**:
  - CPU类型: S7-1200 / 数字输入: 14点 / 数字输出: 10点
  - 工作存储器: 100KB / 通信接口: PROFINET / 电源: DC24V

---

## 中间继电器（Intermediate Relay）

- **中文名称**: 中间继电器 / 辅助继电器 / 小型继电器
- **英文名称**: Intermediate Relay
- **英文缩写**: IR
- **型号示例**: MY4N-J-24VDC-OMRON、RXM4AB2BD-SCHNEIDER、HH54P-24VDC-CHINT
- **安装设备**: 控制柜信号转换、PLC输出扩展、安全回路中间隔离
- **规格参数**:
  - 线圈电压: DC24V / 触点数量: 4NO+4NC / 额定电流: 5A
  - 线圈功耗: 1.6W / 机械寿命: 2000万次 / 电气寿命: 100万次
